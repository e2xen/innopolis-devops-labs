# Logging

Logs monitoring is done with `Grafana`, `Loki`, and `Promtail`.

## Run

Use `docker-compose up` in the current directory, then go to:
- http://localhost:3000 to find Grafana UI
- http://localhost:80 to make a request to the Web App
